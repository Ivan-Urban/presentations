What is SVG?

So, what is SVG?
Scalable Vector Graphics (SVG) is an XML-based markup language for describing two-dimensional vector graphics.
In other words, it is a text file, which is an open web standard for describing two-dimensional vector images.

Advantages and disadvantages

Like everything, SVG has advantages and disadvantages.

Advantages

  • text format
  • scalability
  • high print quality
  • small file size
  • animation
SVG images can be created and edited with any text editor SVG graphics don’t lose any quality if they are zoomed or resized SVG images can be printed with high quality at any resolution SVG images can be animated

Disadvantages

  • cannot support as much detail
  • doesn't work on legacy browsers
SVGs are based on points and paths instead of pixels, they can't display as much detail as standard image formats Legacy browsers, such as IE8 and lower, don't support SVG

How to add SVG to HTML page?

How to add SVG to an HTML page? There are several ways.
Tag <img>
SVG can be added to a page like any other image: via the img tag or as a CSS background.
CSS Background Image 

    {
      width: 200px;
      height: 100px;
      background-image: url('search.svg');
    }
In these cases scripts, animations and other interactivity will not work
Tag <object>
You can also add SVG using the object tag. In this case, you can include stylesheets or scripts.
Inline SVG
Well, since SVG is a text format, you can add it inline to the HTML code. In this way, you can also change styles, add scripts, animations, etc.

Basic figures

There are several basic shapes used for most SVG drawing.
Rectangle
A rectangle can be drawn using the rect tag. Rectangles can also have rounded corners. If no radii are specified, their default values are zero. If one of them is specified, the default value of the other radius is equal to the set one. The fill attribute sets the color inside the figure. Stroke sets the color of the line drawn around the figure, and stroke-width sets the width of that line.
Circle
The circle element draws a circle.
Ellipse
Ellipse is the most common form of the circle element.
Line
The line element takes the positions of two points as parameters and draws a straight line between them. Attribute stroke sets the color of the line.
Polyline
A polyline is a group of connected straight lines. All the points are included in one attribute: points. Each point contains an x and y coordinate, separated by a comma. Different points are separated by spaces. If fill="none" is not set, the lines will join into a filled shape.
Polygon
A polygon is similar to a polyline, in that it is composed of straight lines connecting a list of points. However, for polygons the path automatically connects the last point with the first.

Tag <g>
Tag g is used to group other SVG elements. Any transformations applied to g elements are inherited by its children.

Tag <path>

The path is the most powerful element of the basic SVG forms. It can be used to create all kinds of shapes and Bézier curves. The shape of a path element is defined by one parameter: d.
The d attribute contains a series of commands and parameters used by those commands. Each of the commands is denoted by a specific letter. The M command The L command Command Z
Command H draws a horizontal line.
Command V draws a vertical line.
Bézier curves can be drawn with the C and Q commands. The A command All commands (except Z) exist in two versions: the uppercase command indicates the absolute coordinates on the page, and the lowercase command indicates the relative coordinates (relative to the previous point)
